Github Pagesを使ってウェブサイトを作る

更新日時：2024/01/19 09:30

こんにちは、私です(・∀・)

初ブログがこんなものになってしまうとは思わなかった鯖主ですw (ネタ切れ) まあ、そんなことは置いといて今回はWebを作ったのでてきとーに解説です(?)

必要なもの

  • Webサイトが作れる環境（Windowsとか）
  • プログラミング知識（HTML, CSS, JavaScript）
  • 努力、友情、脚力

プログラミングの知識はわからんでもできます(^.^)

Githubアカウントを作る

Github公式ページにアクセスしてサインアップします

アカウントを持っていればサインインでOKです、 この辺で戸惑う人は一旦「Webを作る」という項目から離れましょう((

この後、 リポジトリを作成する手順に入りますが、 「Webを作る」ということでスキップします。

必要なファイルをリポジトリにぶちこむ

必要なファイルは以下の拡張子のついたものです 
基盤的なもの.html
装飾用.css
アニメーション等.js

よくわかんねえよ！！！って人は「html テンプレート」で検索すると、 ファイルの揃った完成品のファイルをダウンロードできます

自分で作ったものがあるなら、 それを使ったほうが良かったり良くないこともなかったりです((

ファイルを弄り倒す

さっそくファイルをテキストエディタで開きます！ よくhtmlファイルを弄ろうとしてブラウザが立ち上がった！！って人、 よく見ますがテキストエディタで開きましょう((

VSCode Studioが個人的にオススメです

VSCode Studio Web Editor

ひとまず、 これを見てギリ食べ物じゃないことがわかった方から続きを見ましょう！ 文字が良く分からん文字列と記号で挟まれてますが、 英語分からんでもできます(・∀・)b

htmlはタグっていうものに挟まれた文字列をWebに表示する役割があります。 例えば＜p＞と＜/p＞で挟まれた文字列はWebで表示されます！ 最後はスラッシュ(/)を使ってタグの終わりを明記しましょう ＜h1＞を使うと見出し表示になって、 でっかく文字が表示されたりします！

htmlはこんな感じ、 cssに移りますがまずは、 以下のcssコードをコピーして使って見ましょう！

p {
    font-size: 200px;
}

文字から推測できる人もいるかも知れないですが、 このコードを使って読み込むとあら不思議！ pタグを使った文字列が巨大化しました！

他にも色んなタグがありますが、 それぞれに合わせてhtmlやcssを編集できます！ 分からんね。

まとめ

こんな感じで、私はウェブサイトや記事を作っていますがどうでしたか？

感覚や憶測でやっていることも多いのでこの記事は参考までに...

記事ってどうやって締めればいいんだろうね(( それじゃ、さいならーバイ！(*´ω｀*)
